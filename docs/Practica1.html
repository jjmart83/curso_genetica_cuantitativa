1 Introducción

En este ejercicio, estudiaremos la heredabilidad de un rasgo (por ejemplo, longitud de ala) en Drosophila melanogaster, utilizando regresiones simples.

  • mo: Fenotipo (p. ej., longitud de ala) en la madre.
  • fa: Fenotipo en el padre.
  • midparent: Promedio entre mo y fa.
  • off: Fenotipo de la descendencia (por ejemplo, el promedio de la longitud de ala en la progenie).

La pendiente de la regresión off ~ midparent se asocia, en genética cuantitativa, con (h^2). También podemos aproximar h^2 al analizar por separado off ~ mo o off ~ fa.

1.1 Objetivos de la Práctica

  1. Familiarizarse con la manipulación de datos que incluyan mo, fa, midparent, off.
  2. Realizar tres regresiones lineales para estimar la heredabilidad:
    • off ~ midparent
    • off ~ mo
    • off ~ fa
  3. Interpretar las pendientes de regresión y relacionarlos con la heredabilidad (en un escenario simplificado).

3.1.2 Opción B: Simulación de Datos (Ejemplo)

set.seed(123)
n <- 100

# Fenotipos de madre (mo) y padre (fa)
mo <- rnorm(n, mean = 10, sd = 2)
fa <- rnorm(n, mean = 10, sd = 2)

# Calculamos midparent
midparent <- (mo + fa) / 2

# Heredabilidad hipotética
h2 <- 0.4

# Error ambiental
error <- rnorm(n, mean = 0, sd = 1.5)

# off = fenotipo de la descendencia (promedio)
off <- h2 * midparent + error

# Creamos el data frame
datos <- data.frame(mo, fa, midparent, off)

5 Interpretación

6 Discusión de los Resultados

  1. Pendiente:
    • off ~ midparent: la pendiente se asocia a h^2.
    • off ~ mo o off ~ fa: la pendiente se asocia a la mitad de h^2.
  2. Significancia: si la p es < 0.05, la relación es estadísticamente confiable.
  3. Limitaciones: este método asume un modelo muy simple y no corrige efectos maternos ambientales, epistasia, etc.

7 Conclusiones

  • Se introdujo la historia en Drosophila para un rasgo (ej. longitud de ala).

  • Se invita a discutir la plausibilidad de estos resultados y las simplificaciones del modelo.
